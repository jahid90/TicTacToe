# TicTacToe

A C++ implementation of the popular board game - TicTacToe.
The GUI is developed using qt5.

## Development

### Prerequisites to be installed:
```
$ apt get install gcc qt5-default qt5-qmake make
```

### To build the project, run
```
$ ./build.sh
```

### To run the game, run
```
$ ./run.sh
```
